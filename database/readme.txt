### Database ของโครงงานปริญญานิพนธ์เรื่อง การพัฒนาเว็บไซต์และระบบจัดการเนื้อหาภาควิชาคอมพิวเตอร์
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

# อธิบายโครงสร้างของ database
1. โฟลเดอร์ csv
    - จะประกอบไปด้วยไฟล์ csv ของ course = หลักสูตร, personnel = บุคลากร, subject = รายวิชา
2. Dockerfile
    - ใช้สำหรับสร้าง PostgreSQL container ที่ตั้งค่าให้รองรับ UTF-8 และรันไฟล์ "init.sql" อัตโนมัติเพื่อเตรียมฐานข้อมูลตั้งแต่เริ่มต้นใช้งาน
3. init.sql
    - ใช้เก็บฐานข้อมูลต่างๆ เช่น ข่าวสาร, บุคลากร, ผลงานวิจัย, หลักสูตร, รายวิชา, ข่าวการรับสมัคร, ปฏิทินกิจกรรม และผู้ใช้งาน เป็นต้น
4. .env
    - ใช้เก็บ ค่าตั้งค่าของระบบ ประกอบไปด้วย
    4.1) PostgreSQL
        - POSTGRES_DB → ชื่อฐานข้อมูล
        - POSTGRES_USER → username
        - POSTGRES_PASSWORD → password
        - POSTGRES_PORT → พอร์ตของ database
    4.2) pgAdmin
        - PGADMIN_DEFAULT_EMAIL → อีเมลใช้ล็อกอิน
        - PGADMIN_DEFAULT_PASSWORD → รหัสผ่าน
        - PGADMIN_PORT → พอร์ตเข้าใช้งานผ่านเว็บ
    4.3) MinIO
        - MINIO_ROOT_USER → username
        - MINIO_ROOT_PASSWORD → password
5. docker-compose.yml
    - ใช้กำหนดและสั่งรันหลาย ๆ container พร้อมกัน เช่น ฐานข้อมูล PostgreSQL, pgAdmin สำหรับจัดการฐานข้อมูลผ่านเว็บ และ MinIO สำหรับเก็บไฟล์

# วิธีเก็บรูปภาพหรือไฟล์เอกสารใน MinIO
1. เข้าสู่หน้าเว็บไซต์ http://localhost:9001/
2. ดู Username กับ Password ผ่านไฟล์ .env และเข้าสู่ระบบ
3. กด Create Bucket ตั้งชื่อว่า images
4. กด Create new path ตั้งชื่อตามนี้
    4.1 news
    4.2 personnel
    4.3 roadmap

# วิธีตั้งค่าเป็น Public สำหรับ MinIO
1. เปิด visual studio code
2. เปิดโฟลเดอร์ database 
3. เปิด Terminal
4. ใช้คำสั่ง เรียงตามนี้
    4.1 docker exec -it cpsu_minio /bin/sh
    4.2 mc alias set local http://localhost:9000 minioadmin minioadmin123
    4.3 mc anonymous set public local/images
