## อธิบายโครงสร้างของ database
 1. โฟลเดอร์ csv
	- จะประกอบไปด้วยไฟล์ csv ของ course = หลักสูตร, personnel = บุคลากร, subject = รายวิชา
 2. Dockerfile
	 - ใช้สำหรับสร้าง PostgreSQL container ที่ตั้งค่าให้รองรับ UTF-8 และรันไฟล์ `init.sql` อัตโนมัติเพื่อเตรียมฐานข้อมูลตั้งแต่เริ่มต้นใช้งาน
 3. init.sql
	 - ใช้เก็บฐานข้อมูลต่างๆ เช่น ข่าวสาร, บุคลากร, ผลงานวิจัย, หลักสูตร, รายวิชา, ข่าวการรับสมัคร, ปฏิทินกิจกรรม และผู้ใช้งาน เป็นต้น
 4. .env
	- ใช้เก็บ ค่าตั้งค่าของระบบ เช่น ชื่อฐานข้อมูล, ชื่อผู้ใช้, รหัสผ่าน และพอร์ตเข้าใช้งาน เป็นต้น
 5. docker-compose.yml
	 - ใช้กำหนดและสั่งรันหลาย ๆ container พร้อมกัน เช่น ฐานข้อมูล PostgreSQL, pgAdmin สำหรับจัดการฐานข้อมูลผ่านเว็บ และ MinIO สำหรับเก็บไฟล์

## วิธีการใช้งาน
1. เปิด visual studio code
2. เปิดโฟลเดอร์ backend
3. เปิด Terminal
4. ใช้คำสั่ง docker compose up -d เพื่อใช้งาน Database
